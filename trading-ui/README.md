# AetherDesk Prime — Unified Trading Command Surface

AetherDesk Prime is a high-performance, institutional-grade algorithmic trading platform that integrates native execution engines, real-time analytics, and bilingual (AetherDesk/OpenAlgo) design system reactivity.

## Core Features
- **Unified Engine**: Flask-based execution core with Shoonya broker integration.
- **Analytical Intelligence**: Native GEX, IV, and Max Pain analytical engines.
- **Historify System**: DuckDB-powered high-speed historical data logging.
- **Institutional UI**: React + Vite + Shadcn UI with high-density industrial dashboards.
- **Zero-Touch Sync**: Automated build and deployment pipeline via Docker.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS (v4 compatible), Framer Motion.
- **Backend**: Python 3.12 (Flask), Redis, DuckDB, SQLite.
- **Infrastructure**: Docker Compose, Nginx, Supabase (Auth/DB), Kong Gateway.
